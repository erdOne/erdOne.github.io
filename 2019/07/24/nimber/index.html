Posted on 2019-07-24 | Edited on 2019-08-02

最近開始啃IOIC的講義，看到第一章最後一題就開始查資料，最後便生出了這麼一篇文。
本文對遊戲的探討僅限於有限[1]對稱零和完全資訊遊戲。

遊戲

關於遊戲的嚴謹定義，我使用的是Lenstra的定義。

遊戲
一個遊戲就是一個集合。

至於如何遊玩一個集合，我們有如下的定義：

狀態
狀態\(S\)為可以從狀態\(S\)走到的所有狀態的集合。

而輸家就是無法行動，亦即拿到的集合是\(\varnothing\)。
於是有限的遊戲就是所有繼承有限集合（Hereditarily finite set），也就是所有屬於\(V(\omega)\)的集合[2]。

舉個例子，一個個數為\(3\)的Nim[3]（寫作\(Nim(3)\)）就是\(\{\varnothing,\ \{\varnothing\},\ \{\varnothing,\ \{\varnothing\}\}\}\) [4]。
稍微看過集合論的讀者會發現這就是馮諾伊曼基數。

接著定義兩個集合：

後手擁有必勝策略（\(\mathcal{P}\)）
\(S \in \mathcal{P} \Longleftrightarrow \forall s\in S,\ s \not\in\mathcal{P}\)

先手擁有必勝策略（\(\mathcal{N}\)）
\(S \in \mathcal{N} \Longleftrightarrow \exists s\in S,\ s \not\in\mathcal{N}\)

你會發現\(\mathcal{P}\)與\(\mathcal{N}\)構成\(V(\omega)\)一個劃分，亦即所有集合不屬於\(\mathcal{P}\)就屬於\(\mathcal{N}\)。
另外由定義也可以看出\(\varnothing\in\mathcal{P}\)，符合上文的描述。

遊戲和

有了以上的定義，我們便可以開始定義遊戲的算術。

遊戲和
\(S_1+S_2 :=\{s + S_2|s\in S_1 \}\cup\{S_1 + s|s\in S_2 \}\)

用白話文來說，就是將兩個遊戲一起玩，每次選擇一個遊戲並進行一次行動。
你可以簡單的利用結構歸納法驗證這個定義符合結合律（\(S_1+(S_2+S_3)=(S_1+S_2)+S_3\)）與交換律（\(S_1+S_2=S_2+S_1\)）
於是\((V(\omega), +)\)構成一個交換么半群（commutative monoid）。

而有了加法，我們便可以定義外延意義下的等價。

遊戲等價
\(S_1\sim S_2 \Longleftrightarrow \forall S,\ S_1+S\in\mathcal{P} \Leftrightarrow S_2+S\in\mathcal{P}\)

也就是如果不論加上任何遊戲，輸贏情形都相同，則這兩個遊戲等價。
你可以簡單的檢驗他符合等價的定義：
自反性（\(S\sim S\)）、對稱性（\(S_1\sim S_2 \Longleftrightarrow S_2\sim S_1\)）、遞移性（\(S_1\sim S_2 \land S_2\sim S_3 \Longrightarrow S_1\sim S_3\)）

由以上的定義，你不難發現以下事情

\(s\in\mathcal{P}\Longleftrightarrow S + s \sim S\)

以及

\(S+S\in\mathcal{P}\)

由以上觀察你可以發現，\((V(\omega)/\sim,+,\mathcal{P})\)事實上構成一個指數為\(2\)的交換群。

以上定義就是為了帶出斯普萊格–格隆第定理：

Sprague-Grundy theorem
\(\forall S\ \exists n\ [S\sim Nim(n)]\)

也就是所有遊戲等價於一個Nim遊戲。

證明如下：
利用結構歸納法，若已知\(S_i\sim Nim(a_i)\)，則我們可以證明
\[S=\{S_1,\dots,S_k\}\sim S'=\{Nim(a_1),\dots Nim(a_k)\}\sim Nim(m)\]
其中\(m=\operatorname{mex}\{a_1,\dots,a_k\}\)指的是minimum excluded value，也就是不在此集合裡面的最小的非負整數。

首先考慮第一個等價：
\(S\sim S'\)
\(\Leftrightarrow S+S'\in\mathcal{P}\)
\(\Leftrightarrow \forall s\in S+S'[s\in\mathcal{N}]\)
\(\Leftrightarrow \forall s\in S+S'\big[\exists t\in s[t\in\mathcal{P}]\big]\)
考慮\(s\)的形式。
若形如\(S+Nim(a_i)\)則取\(t=S_i+Nim(a_i)\)；
若形如\(S_i+S'\)則一樣取\(t=S_i+Nim(a_i)\)。
由歸納法前提得知\(t\in\mathcal{P}\)。

考慮第二個等價：
\(S'\sim Nim(m)\Leftrightarrow \forall s\in S'+Nim(m)\big[\exists t\in s[t\in\mathcal{P}]\big]\)
一樣考慮\(s\)的形式：
若形如\(Nim(a_i)+Nim(m)\)，則根據\(m\)的定義，\(a_i\neq m\)，故可以取\(t = Nim(\min\{a_i,m\}) + Nim(\min\{a_i,m\})\)；
若形如\(S'+ Nim(m')\)，則由\(m\)的定義得知對於任何\(m'<m\)都有\(Nim(m')\in S'\)，故取\(t = Nim(m') + Nim(m')\)即可。

起始條件是\(\varnothing=Nim(0)\)，由結構歸納法得證。

由以上定理證明，可以自然而然定義出一個遊戲的\(SG\)值，亦即跟此遊戲等價的Nim的石頭個數。

Sprauge-Grundy value
\(SG(S) := \operatorname{mex}\left\{SG(s)|s\in S\right\}\)

於是我們有一個\(V(\omega)\)至非負整數的雙射。

Sprauge-Grundy theorem
\(SG(S_1+S_2) = SG(S_1)\oplus SG(S_2)\)

若\(k<l-1\)，則
\(2^{2^l - 1} \otimes x\)
$ = 2^{2^{l-1} - 1} \otimes 2^{2^{l-1}} \otimes x$
$ = (2^{2^{l-1} - 1} \otimes x)\otimes 2^{2^{l-1}}$

若\(k = l-1\)則
\(2^{2^l - 1} \otimes x\)
$ = 2^{2^{l-1} - 1} \otimes 2^{2^{l-1}} \otimes (a\otimes 2^{2^{l-1}} \oplus b)$
$ = 2^{2^{l-1} - 1} \otimes ( a\otimes 2^{2^{l-1}} \otimes 2^{2^{l-1}} \oplus 2^{2^{l-1}} \otimes b))$
$ = 2^{2^{l-1} - 1} \otimes ( a\otimes 2^{2^{l-1}} \oplus 2^{2^{l-1}} \otimes b \oplus 2^{2^{l-1}-1} \otimes a)$
$ = 2^{2^{l-1} - 1} \otimes ( (a\oplus b)\otimes 2^{2^{l-1}} \oplus (2^{2^{l-1}-1} \otimes a))$
$ = (a\oplus b)\otimes 2^{2^{l-1}} \oplus (2^{2^{l-1} - 1} \otimes 2^{2^{l-1}-1} \otimes a)$

若\(k > l - 1\)則
\(2^{2^l - 1} \otimes x\)
$ = 2^{2^l - 1} \otimes (a\otimes 2^{2^k} \oplus b)$
$ = ((2^{2^l - 1} \otimes a)\otimes 2^{2^k} \oplus (2^{2^l - 1} \otimes b))$
